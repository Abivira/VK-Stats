# VK-Stats
Virat Kohli Stats
## How to Run This Project Locally

### Install Node.js
Download and install Node.js (LTS) from:
https://nodejs.org/

After installation, verify:
node -v
npm -v

### Clone the Repository
git clone https://github.com/YOUR_USERNAME/virat-kohli-stats.git
cd virat-kohli-stats

### Install Dependencies
npm install

### Run the Project
npm run dev

### Open in Browser
Open the URL shown in the terminal (usually):
http://localhost:5173
